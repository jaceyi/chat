# Chat

基于 Draft-js 实现的富文本输入框。

## 开发

```shell
yarn
yarn start
yarn build
```

本地预览，先参考 [Firebase CLI](https://firebase.google.com/docs/cli?authuser=0#install_the_firebase_cli) 并安装。
```shell
firebase emulators:start
```
